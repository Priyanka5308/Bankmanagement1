Smart Bank Management System

A web-based banking application developed using Java, Spring MVC, Spring ORM (JPA), JSP, and MySQL. This project allows customers to securely manage bank accounts, transfer funds, view transaction history, and apply for loans through an online platform. It also includes an admin module for managing customers, approving loan requests, and generating reports.

**Features**
  * User Registration & Secure Login
  * Account Dashboard
  * Fund Transfer using Account Number / UPI
  * Mini Statement (Last 10 Transactions)
  * Loan Application & Tracking
  * Admin Panel for Customer & Loan Management
  * Transaction Reports
**Tech Stack**
  * Java
  * Spring MVC
  * Spring ORM (JPA)
  * JSP
  * HTML, CSS
  * MySQL
  * Apache Tomcat

The main objective of this project is to simplify traditional banking operations by providing a secure and efficient online banking system with customer and admin functionalities.
